# 摄影师个人作品集网站

## 项目概述
为摄影师打造一个富有艺术感和互动性的个人作品集网站，展示作品、建立个人品牌。采用现代简约设计，完美适配各种设备。

## UI/UX流程
- 响应式导航栏，支持移动端适配
- 全屏幻灯片展示作品，增加直观 CTA 按钮
- 瀑布流作品集布局
- 个性化关于我页面
- 简洁的联系表单

## 技术架构
- 前端：React + TailwindCSS + Framer Motion
- 路由：React Router Dom
- 动画：Swiper + Framer Motion
- 响应式设计：TailwindCSS
